# Тату-салон

Это репозиторий для сайта тату-салона. Он содержит HTML и SCSS файлы для фронтенда и PHP файлы для функционала записи на обучение.

## Инструкции по запуску

1. Клонируйте репозиторий на вашу локальную машину с помощью следующей команды:
git clone <URL репозитория>
Замените `<URL репозитория>` на фактический URL вашего репозитория.

2. Откройте терминал и перейдите в корневую папку проекта.

3. Для стилей сайта используется SCSS, поэтому перед первым запуском необходимо скомпилировать файлы CSS. Выполните следующую команду:
sass --watch styles.scss:styles.css
Это позволит автоматически компилировать SCSS в CSS при внесении изменений.

4. Вам понадобится локальный сервер для запуска PHP. Если у вас уже есть сервер, перейдите к следующему шагу. Если нет, вы можете использовать встроенный сервер PHP, запустив следующую команду:
php -S localhost:tattoo

5. Теперь откройте веб-браузер и перейдите по следующему адресу:
http://localhost:tattoo
Вы должны увидеть ваш сайт тату-салона.

## Дополнительная информация

- Для работы с записью на обучение, убедитесь, что ваш сервер поддерживает PHP и имеет необходимые расширения (например, MySQLi для работы с базой данных).
- Внесите соответствующие изменения в PHP файлы, чтобы они соответствовали вашей системе управления записями на обучение.
- При необходимости настройте базу данных и укажите соответствующие данные для подключения в PHP файле, отвечающем за запись на обучение.
- Обратите внимание, что в этом репозитории предоставлены только основные файлы. Вы можете добавить дополнительные файлы, изображения, шрифты и т. д., по мере необходимости.
